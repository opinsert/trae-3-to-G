# 百度在线OCR服务集成计划

## 📋 概述

为解决当前OCR识别效果差的问题，计划将项目的OCR引擎从本地Tesseract替换为百度在线OCR服务。

### 百度OCR优势

* **免费额度充足**：每月1000次免费调用，个人使用完全足够

* **识别准确率高**：基于百度AI技术，中文识别准确率超过99%

* **无需下载模型**：云端处理，不占用本地存储

* **API调用简单**：提供完善的Python SDK

## 📝 实施步骤

### 步骤1：注册百度智能云账号并获取API密钥

1. **访问百度智能云官网**

   * 网址：<https://cloud.baidu.com/>

   * 使用百度账号登录（或注册新账号）

2. **创建文字识别应用**

   * 进入控制台 → 产品服务 → 人工智能 → 文字识别

   * 点击"创建应用"

   * 填写应用信息（应用名称、应用描述随意填写）

   * 在"安全认证"中获取：

     * `App ID`

     * `API Key`

     * `Secret Key`

3. **获取免费额度**

   * 实名认证后自动获得每月1000次免费调用额度

### 步骤2：安装百度AIP SDK

在后端环境中安装百度OCR的Python SDK：

```bash
pip install baidu-aip
```

### 步骤3：配置API密钥

将获取的API密钥配置到项目中：

1. **修改环境变量文件**（`backend/.env`）

   ```
   BAIDU_APP_ID=您的App_ID
   BAIDU_API_KEY=您的API_Key
   BAIDU_SECRET_KEY=您的Secret_Key
   ```

2. **更新配置加载模块**（`backend/app/utils/config.py`）

   * 添加百度API密钥的读取逻辑

### 步骤4：实现百度OCR识别模块

创建百度OCR处理器：

**文件位置**：`backend/app/core/baidu_ocr.py`（可选新文件）

**实现内容**：

1. 初始化AipOcr客户端
2. 实现图片识别方法
3. 实现结果解析和格式化

### 步骤5：修改现有OCR处理逻辑

更新`backend/app/core/ocr_processor.py`：

1. **保留Tesseract作为备用方案**

   * EasyOCR（失败）

   * 百度OCR（主方案）

   * Tesseract（最终备用）

2. **添加百度OCR识别逻辑**

   ```python
   def _run_baidu_ocr(self, image_data):
       """调用百度OCR API进行识别"""
       # 调用百度API
       # 解析返回结果
       # 返回识别的文本
   ```

3. **修改recognize方法**

   * 优先尝试EasyOCR

   * 如果失败，尝试百度OCR

   * 最后使用Tesseract

### 步骤6：更新错误处理和日志

1. **增强错误处理**

   * 处理API调用失败

   * 处理网络超时

   * 处理免费额度用尽

2. **添加详细日志**

   * 记录使用的OCR引擎

   * 记录API调用结果

   * 记录识别置信度

### 步骤7：测试OCR识别功能

1. **本地测试脚本测试**

   ```bash
   cd backend
   python test_ocr.py test.jpg
   ```

2. **启动前后端服务测试**

   * 后端：`uvicorn app.main:app --host 0.0.0.0 --port 5173 --reload`

   * 前端：`cd frontend && npm run dev`

3. **前端界面测试**

   * 上传工序卡图片

   * 点击识别按钮

   * 检查识别结果

## ⚠️ 注意事项

1. **API密钥安全**

   * 不要将API密钥提交到GitHub

   * 使用环境变量存储敏感信息

   * 定期检查使用量

2. **网络要求**

   * 百度OCR需要互联网连接

   * 无法在离线环境使用

   * 考虑添加本地备用方案

3. **数据安全**

   * 图片数据会上传到百度服务器

   * 如有保密需求，考虑私有化部署

   * 确认百度数据安全合规

4. **成本控制**

   * 监控免费额度使用情况

   * 设置额度提醒

   * 避免异常调用
